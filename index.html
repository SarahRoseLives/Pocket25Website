Capabilities

Built for Enthusiasts

Native RTL-SDR Support

No external driver apps required. Pocket25 communicates directly with your RTL-SDR dongle via USB OTG for low-latency performance.

RadioReference Import

Log in to your RadioReference account and import local trunked systems, sites, and talkgroups directly into the internal database.

GPS Site Hopping

Driving across the county? Pocket25 uses your GPS location to automatically switch to the nearest tower site in a trunked system.

Smart Filtering

Easily toggle between Whitelist and Blacklist modes. Long-press any talkgroup in the recent list to instantly mute unwanted traffic.

Live DSD Logs

View the raw output from the embedded DSD decoder. Watch P25 sync bits, TSBK data, and error correction stats in real-time.

Remote RTL-TCP

Don't have a dongle handy? Connect to a remote RTL-TCP server over WiFi and stream raw IQ data for processing.

System Requirements

  • Android Device Requires USB Host (OTG) support. Optimized for landscape orientation.
  • Hardware RTL-SDR V3 or compatible dongle + USB OTG Cable.
  • Network (Optional) Required for RadioReference imports and Remote RTL-TCP connections.
